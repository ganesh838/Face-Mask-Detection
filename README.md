# 😷 Face Mask Detection using Deep Learning

This project uses a CNN model to detect whether a person is wearing a face mask.

## Features
- Image upload/capture image 
- Mask / No Mask prediction
- Confidence score
- Interactive FAQ
- Streamlit-based UI

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
